## Getting Started

1. **Sign up on [[Netlify]](https://www.netlify.com).**
2. **Connect your [[GitHub]] repository.**
3. **Deploy your project with one click.**
4. **Use the [[Netlify]] CLI for advanced workflows.**
   ```sh
   npm install -g netlify-cli  
   netlify login  
   netlify deploy  
   ```
